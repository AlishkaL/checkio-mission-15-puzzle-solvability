П'ятнашки, "Гра в 15" (15-puzzle) це досить стара і відома головоломка, яка мала власний бум інтересу!

У цій місії тобі дано розташування пластинок position - {% if is_js %}масив з 4 масивів{% else %}список з 4 списків{% endif %} однакової довжини з усіма цілими числами від 1 до 16 включно в довільному порядку (але число 16 завжди символізує порожнє місце!). Твоя функція повинна повернути логічне (булеве) значення, чи ця комбінація пластинок може бути розв'язана (може бути перетворена дозволеними ходами у "звичайний" порядок) або ні, відповідно як {% if is_js %}t{% else %}T{% endif %}rue or {% if is_js %}f{% else %}F{% endif %}alse.

Ось цікаве відео про саму головоломку та проблему її розв'язаності з алгоритмом перевірки. Тож якщо ти не любиш спойлери, то можете спробувати спочатку вирішити місію, не переглядаючи відео.

Вхідні дані: Розташування пластинок як{% if is_js %}масив масивів{% else %}список списків{% endif %}.

Вихідні дані: Логічне (булеве) значення.
